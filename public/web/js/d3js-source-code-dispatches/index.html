d3.js源码分析——Dispatches

2016-09-30
Web开发 D3.js
  1. d3.dispatch
    1. dispatch.on(typenames[, callback])

d3的dispatch模块是对原生事件处理的封装，通过该模块可以注册自定义的事件并绑定回调函数。

d3.dispatch

该模块用于注册自定义名称的回调函数，并且可以调用这些函数。

将传入的事件类型存入dispatch对象中。

dispatch.on(typenames[, callback])

用于将事件和回调函数进行绑定。
